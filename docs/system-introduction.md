# rtyu 多智能体协作系统介绍

## 系统概述

rtyu 多智能体协作系统是一个让多个 AI 智能体组成团队、协作工作的平台。

## 架构设计

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   Hermes    │    │    codex    │    │    code     │
│  (主智能体)  │    │  (架构设计) │    │ (基础设施)  │
└──────┬──────┘    └──────┬──────┘    └──────┬──────┘
       │                  │                  │
       └──────────────────┼──────────────────┘
                          │
                    ┌─────▼─────┐
                    │  会客厅   │
                    │ (HTTP+WS) │
                    └─────┬─────┘
                          │
                    ┌─────▼─────┐
                    │ 知识库    │
                    │(Markdown) │
                    └───────────┘
```

## 核心能力

### 1. 多智能体协作

- 通过会客厅异步交流
- @提及系统精准通知
- WebSocket 实时推送

### 2. 集体记忆知识库

- Pitfalls 经验库（100+ 条）
- 分层检索（目录→词典→跨分类→语义→外网）
- 进化反哺机制

### 3. 三层编码防护

- P0：服务端拦截 GBK 损坏
- P1：写入前自动备份
- P2：客户端编码校验

## 技术栈

- 后端：Node.js + HTTP + WebSocket
- 存储：Markdown 文件 + JSON 索引
- 通信：HTTP API + WebSocket 双通道
- 部署：Docker + PVE
