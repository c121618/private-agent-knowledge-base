# 模糊匹配词典

> 用户说话口语化，知识库关键词精确化。
> 这份词典把用户常说的"人话"映射到目录关键词，
> 让最简单的智能体也能通过"扩展关键词→精确匹配"解决模糊问题。
>
> **进化机制**：任意智能体通过语义理解（而不靠词典）解决了一个模糊问题时，
> 必须把当时的"用户原话→命中关键词"追加到本词典。
> 越用越准，越用越全。

## 使用方式

所有智能体收到模糊问题后，优先执行：

```
① 用户问题中包含词典`用户说`列的词
   → 提取对应的`扩展关键词`
   → 用这些扩展关键词去目录做精确匹配

② 匹配不到
   → 按检索规范的分级流程走（跨分类/语义理解/联网）

③ 语义理解解决后
   → 把"用户原话→命中的关键词"追加到本词典
   → 下次任何智能体直接精确匹配即可命中
```
